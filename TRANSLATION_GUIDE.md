# PDF 閱讀器 - 翻譯功能使用指南

## 功能概述

PDF 閱讀器現在支援自動翻譯功能，使用 Google Translate 翻譯引擎（透過 deep-translator 套件）。

## 主要功能

### 1. 翻譯選取的文字

**操作步驟：**
1. 在 PDF 中用滑鼠拖曳選取要翻譯的文字區域
2. 選取後會自動切換到側邊欄的「翻譯」分頁
3. 選取的文字會顯示在「原文」區域
4. 點擊「翻譯選取」按鈕開始翻譯
5. 翻譯結果會顯示在「譯文」區域

**快捷鍵：** `Ctrl+T` （翻譯選取文字）

### 2. 翻譯目前頁面

**操作步驟：**
1. 打開要翻譯的 PDF 文件
2. 選擇「工具」→「翻譯目前頁面」
3. 系統會自動提取目前頁面的所有文字並翻譯
4. 翻譯結果顯示在側邊欄的翻譯面板

### 3. 翻譯整份文件

**操作步驟：**
1. 打開要翻譯的 PDF 文件
2. 選擇「工具」→「翻譯整份文件」
3. 確認翻譯對話框
4. 系統會逐頁翻譯整份文件
5. 進度條會顯示翻譯進度
6. 完成後所有頁面的翻譯結果會合併顯示

**注意：** 翻譯大型文件可能需要較長時間，請耐心等待。

### 設定來源和目標語言

在翻譯面板中：
- **來源語言：** 可選擇「自動偵測」或指定語言
- **目標語言：** 選擇要翻譯成的語言（預設：繁體中文）

## 網路需求

翻譯功能使用 Google Translate API，因此需要網路連線才能使用。翻譯時會即時連線到 Google 伺服器進行翻譯。

## 使用技巧

1. **精確選取文字：** 使用滑鼠仔細拖曳選取要翻譯的文字區域，確保選取範圍準確
2. **分段翻譯：** 對於長文件，建議分頁或分段翻譯，獲得更好的翻譯品質
3. **檢查翻譯結果：** 機器翻譯可能不完美，建議核對重要內容
4. **網路連線：** 確保有穩定的網路連線以獲得最佳翻譯體驗

## 系統需求

- Python 3.8 或更高版本
- deep-translator >= 1.11.0
- 穩定的網路連線（使用時需要連線到 Google Translate）

## 安裝

確保已安裝所需的依賴套件：

```bash
pip install -r requirements.txt
```

這會自動安裝 deep-translator 和相關依賴。

## 疑難排解

### 翻譯失敗

**問題：** 翻譯時出現錯誤訊息

**解決方法：**
1. 檢查網路連線
2. 確認已正確安裝 deep-translator
3. 檢查防火牆設定，確保可以連線到 Google 服務
4. 重新啟動應用程式

### 網路連線問題

**問題：** 無法連線到翻譯服務

**解決方法：**
1. 檢查網路連線狀態
2. 確認防火牆設定允許連線到 translate.google.com
3. 如果在中國大陸，可能需要使用 VPN

### 翻譯速度慢

**問題：** 翻譯大型文件速度很慢

**解決方法：**
1. 這取決於網路速度和文件大小
2. 考慮分頁或分段翻譯
3. 確保網路連線穩定

## 已知限制

1. **網路需求：** 需要網路連線才能使用翻譯功能
2. **翻譯品質：** 翻譯品質取決於 Google Translate，對於專業術語可能不夠準確
3. **速度限制：** 翻譯速度受網路速度影響
4. **使用限制：** Google Translate 有使用頻率限制，過度使用可能被暫時封鎖

## 技術說明

本翻譯功能使用 `deep-translator` 套件，這是一個輕量級的 Python 翻譯庫，支援多種翻譯引擎。目前使用 Google Translate 作為翻譯引擎。

### 優點
- 輕量級，安裝快速
- 無需下載大型語言模型
- 支援多種語言
- Google Translate 品質較好

### 缺點
- 需要網路連線
- 受 Google Translate 服務限制
