## 🚀 如何開始使用

### 方法一：Windows 快速啟動（推薦）

```
1. 雙擊 install.bat    → 自動安裝
```

### 方法二：命令列啟動

```bash

# 2. 安裝依賴（首次使用）
pip install -r requirements.txt

# 3. 啟動應用程式
python main.py
```
