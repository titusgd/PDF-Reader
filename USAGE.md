# 使用指南

## 快速開始

### 啟動應用程式

1. 啟動虛擬環境
2. 執行 `python main.py`
3. 應用程式視窗將會開啟

### 開啟 PDF 檔案

方法 1: 選單列 → 檔案 → 開啟 (Ctrl+O)
方法 2: 工具列 → 點擊「開啟」按鈕

## 主要功能

### 1. 基本導航

#### 頁面切換
- **上一頁**: 點擊工具列的「上一頁」按鈕
- **下一頁**: 點擊工具列的「下一頁」按鈕
- **跳轉到指定頁**: 在工具列的頁碼輸入框中輸入頁碼

#### 側邊欄
- **縮圖檢視**: 點擊側邊欄的「縮圖」分頁，點擊縮圖可快速跳轉
- **切換側邊欄**: 工具列 → 點擊「側邊欄」按鈕

### 2. 縮放控制

- **放大**: Ctrl + +（加號）或工具列的「放大」按鈕
- **縮小**: Ctrl + -（減號）或工具列的「縮小」按鈕
- **適應寬度**: 選擇縮放下拉選單中的「適應寬度」
- **適應頁面**: 選擇縮放下拉選單中的「適應頁面」
- **自訂縮放**: 從下拉選單選擇 50%、75%、100%、125%、150%、200%、300%

### 3. 旋轉頁面

- **逆時針旋轉**: 工具列 → 「逆時針旋轉」按鈕
- **順時針旋轉**: 工具列 → 「順時針旋轉」按鈕

### 4. 搜尋功能

1. 按下 Ctrl+F 或選單 → 編輯 → 搜尋
2. 輸入要搜尋的文字
3. 點擊「確定」
4. 系統會跳轉到第一個搜尋結果並顯示總結果數

### 5. 註解工具

#### 啟用註解工具列
選單 → 工具 → 註解工具

#### 可用的註解工具

**高亮標記**
1. 點擊註解工具列的「高亮」按鈕
2. 在 PDF 頁面上拖曳選取要高亮的區域
3. 釋放滑鼠，高亮註解將被新增

**底線**
1. 點擊「底線」按鈕
2. 選取要加底線的區域

**刪除線**
1. 點擊「刪除線」按鈕
2. 選取要加刪除線的區域

**文字註解**
1. 點擊「文字註解」按鈕
2. 在頁面上點擊要新增註解的位置
3. 在對話框中輸入註解內容
4. 點擊「確定」

**手繪**
1. 點擊「手繪」按鈕
2. 在頁面上自由繪製

**圖形註解**
1. 點擊「矩形」、「圓形」或「箭頭」按鈕
2. 在頁面上拖曳繪製圖形

**返回選擇模式**
點擊「選擇」按鈕

### 6. 書籤管理

#### 新增書籤
1. 瀏覽到要加入書籤的頁面
2. 按 Ctrl+B 或選單 → 工具 → 新增書籤
3. 輸入書籤標題和描述（可選）
4. 點擊「確定」

#### 使用書籤
1. 點擊側邊欄的「書籤」分頁
2. 雙擊書籤即可跳轉到該頁面

#### 刪除書籤
1. 在書籤列表中選擇要刪除的書籤
2. 點擊「刪除」按鈕

### 7. 表單編輯

1. 開啟包含表單的 PDF
2. 選單 → 工具 → 表單編輯
3. 系統會顯示文件中的表單欄位數量
4. 可以直接在 PDF 中填寫表單欄位

### 8. 數位簽章

1. 選單 → 工具 → 新增簽章
2. 輸入簽章文字
3. 簽章會被新增到當前頁面的中心位置
4. 可以手動調整簽章位置（功能開發中）

### 9. 儲存檔案

#### 覆蓋原檔案
- Ctrl+S 或選單 → 檔案 → 儲存

#### 另存新檔
- Ctrl+Shift+S 或選單 → 檔案 → 另存新檔
- 選擇儲存位置和檔案名稱

### 10. 列印

1. Ctrl+P 或選單 → 檔案 → 列印
2. 列印功能目前正在開發中

### 11. 檢視模式

#### 全螢幕模式
- 按 F11 進入/退出全螢幕模式

#### 深色模式
- 工具列 → 點擊「深色模式」按鈕
- 或使用選單切換

## 進階功能

### 批次處理註解

1. 使用註解工具新增多個註解
2. 儲存檔案以保留所有註解

### 匯出書籤

書籤會自動儲存在應用程式設定中，下次開啟相同檔案時會自動載入。

### 自訂設定

應用程式會自動記住：
- 視窗大小和位置
- 最後使用的縮放級別
- 深色模式偏好設定
- 最近開啟的檔案（功能開發中）

## 故障排除

### 應用程式無法啟動
1. 確認虛擬環境已啟動
2. 確認所有依賴已安裝
3. 檢查是否有錯誤訊息

### PDF 無法開啟
1. 確認檔案是有效的 PDF 格式
2. 確認檔案沒有密碼保護或損壞
3. 嘗試用其他 PDF 閱讀器開啟以驗證檔案

### 註解無法儲存
1. 確認檔案沒有被其他程式鎖定
2. 確認有檔案寫入權限
3. 使用「另存新檔」儲存到不同位置

### 效能問題
1. 大型 PDF（100+ 頁）可能需要較長載入時間
2. 降低縮放比例可以提升渲染速度
3. 關閉不必要的註解工具列
